# Retail Sales Dashboard Analysis

## Overview
This project involves the analysis of retail sales data to understand sales trends, customer behavior, and product performance. The insights gained can be used to inform business strategies and improve sales performance.

## Data Source
The analysis is based on the `retail_sales_dataset.csv` file, which contains various transaction details including date, customer information, product category, quantity, price, and total amount.

## Analysis Performed
- **Data Loading**: The retail sales data is loaded into a pandas DataFrame.
- **Data Cleaning**: Checked for missing values and duplicates, ensuring data quality.
- **Data Transformation**: Mapped product categories to numerical representations for potential further analysis.
- **Key Metrics Calculation**: Calculated total sales, average sales, maximum/minimum sales, and total orders.
- **Aggregated Sales Data**: Analyzed sales and quantity distribution by product category and gender.
